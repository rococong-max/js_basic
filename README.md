## 변수
* 프로그래밍의 첫번째 작업은 필요한 데이터들을 반드시 **변수(저장소)**로 저장해야한다.
## 변수 작성법
* `변수선언키워드 생성변수명;` -> 빈 저장소 생성
    * `var box;`
* `변수선언키워드 생성변수명 대입연산자 대입값;`
    * `var box = 1;`
* `생성변수명 대입연산자 대입값;`
    * `box = 2`
* 변수선언키워드 : var, let, const
* 생성변수명 - 영문, 숫자조합 (숫자는 영문뒤로)
## 변수 작성 시 주의사항
* 변수의 이름은 문자 (a~z, A~Z),숫자 (0~9), 달러($), 밑줄(_)이 포함됨
* 변수의 이름은 숫자로 시작할 수 없음
* 변수의 이름은 대시(-)나 마침표(.) 기호를 사용할 수 없음
* 키워드나 예약어는 사용할 수 없음
    * 키워드, 에약어 종류 : delete, this, false, ture, if, default, var, new, class 등
* 모든 변수는 대/소문자를 구별함 (score와 Score는 다른 변수!)
* **변수가 저장할 정보의 종류를 표현하는 의미있는 단어를 사용해야 함**

## 데이터타입 구분 함수 `typeof()`
* css의 배경색상으로 영역구분하는 것처럼 데이터타입 구분목적 사용
1. 특정 데이터타입을 담아둔 변수를 생성
    * `let num = 1`
    * `let str = '1'`
2. 해당 변수의 데이터타입을 구분하기위해 typeof() 함수확인
    * `console.log(typeof(num));` //Number(숫자)
    * `console.log(typeof(str));` //String(문자)
## 산술연산자
* 피연산자가 숫자인 경우
* a+b 더하기
* a-b 빼기
* **a*b 곱하기**
* **a/b 나누기**
* **a%b 나머지** : 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 나머지
* a**b 거듭제곱 : 왼쪽의 피연산자를 오른쪽 피연산자 값만큼 곱한 값
* 계산할 수 없는 경우는 NaN 처리 ex)'two'*2 = NaN
## 함수 종류
### 내장함수
* `prompt('질문')` : 질문 답변 창 띄우는 함수, 문자열로 인식
    * `input` 태그로 입력받는 사용자입력값도 모두 문자로 처리
    * 쇼핑몰 수량 `input`도 문자로 기본적으로 인식됨
* `Number(숫자로변화하고싶은데이터 또는 함수)`
* prompt(Number()) **X** prompt 질문입력위치에 다른 함수 작성
* Number(prompt()) **O** Number괄호안은 변환대상 prompt함수 작성 가능
* `toLocaleString()` : 숫자를 특정 국가 기호에 맞게 문자열로 바꿔주는 함수
    * 문자로 바꿔주는 함수이므로 반드시 모든 계산이 끝난 마지막에 적어야 됨
### 사용자 정의 함수 function
* `function 함수명(전달인자){함수실행내용}`
    * function : 함수 선언 시 함수라는 것을 나타내는 키워드
    * 함수이름 : 개발자가 만든 함수의 의미있는 이름작성
    * 전달인자 : 매개변수, 인수 등 여러가지 용어를 사용, 함수 내에 데이터를 전달하는 변수명이라 보면 되며 전달 데이터가 많을 땐 객체 타입으로 전달하기도
    * 중괄호{} : 함수 영역의 시작과 끝을 낱내어 함수의 반복적인 실행 내용을 작성
## 복합대입연산자
* 연산자 | 예시 | 예시뜻 | 예시풀이 순서
* `+=` `a += b` `a = a + b` 기존 a값에 b를 더하여 a에 대입한다.
* `-=` `a -= b` `a = a - b` 기존 a값에 b를 빼서 a에 대입한다.
* `*=` `a *= b` `a = a * b` 기존 a값에 b를 곱해서 a에 대입한다.
* `/=` `a /= b` `a = a / b` 기존 a값에 b를 나눠서 a에 대입한다.
* `%=` `a %= b` `a = a % b` 기존 a값에 b를 나눈 나머지값을 a에 대입한다.

## 배열 array
* 여러 개의 값(데이터)을 순서대로 저장할 수 있는 자료 구조
* []대괄호를 사용해서 각 데이터를 저장
* 각 개별 값은 인덱스(index)로 접근, 숫자는 0부터 인식
* 데이터 전체 길이(length)는 1부터 인식
* (예시) `let 변수명 [데이터1,데이터2,데이터3]`
