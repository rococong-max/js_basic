## 제이쿼리 기본 문법
* `$()` 제이쿼리 기본 선언법
* `$('.클래스명')`
* `$('#아이디명')`
* `$('태그명')`
* `''` 따옴표 안 CSS 선택자처럼 원하는 대상 작성 가능
* `$('대상').show();` 대상 보이기
* `$('대상').hide();` 대상 숨기기
* `$('대상').css('속성', '값');` CSS 적용하기
* `$('대상').removeClass();` class 제거하기 (괄호 비우면 모든 클래스 제거)
* `$('대상').addClass('추가클래스명');` class 적용하기 추가클래스명에는 `.`생략 작성함
## -webkit-
* -webkit- 파이어폭스, 크롬 등 브라우저에서 안전하게 속성 적용하는 웹브라우저 엔진 명령어
* 글자 안에서 그라데이션 효과 및 배경이미지를 마스크씌워서 보여주고 싶을 때 사용하는 속성값 모음 (4개 고정 세트로 사용)
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
    background-clip: text;
    color: transparent;
# 자바스크립트 26/03/06
* 목표 설정 -> 흐름도 제작 -> 각 단계를 코드로 표현
* object : 객체, 사용자가 컨트롤 할 수 있는 객체대상
    * (예)html의 태그
* property : 속성, 개체가 가지는 속성
* event : 이벤트, 인터넷 블우저에서 발생할 수 잇는 사건
    * (예)클릭, 그래그, 더블클릭,키보드 입력 등
* 실행
    * (예)확인,삭제,취소,이동 등
    * function : 함수
    * method : 메소드
    * 클래스 내부에 선언된 함수는 메소드라 부르며, 클래스와 관련없다면 일반적으로 전부 함수라고 부른다.
* **A에 마우스를 올렸을 때 B가 나타난다 == 객체에 이벤트했을 때 객체 메소드(함수) 실행**
* async와 defer 차이점
    * async : 패이지 로드 관계업이 다운로드 후 즉시 실행 스크립트
    * defer : 페이지로드가 완료된 후 즉시 실행, 외부 스크립트 시에만 사용 가능
